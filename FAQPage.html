What is the Beta Key? What do I do with it?

When you create an app, you will receive a rate limited beta key, which we advise you to use in development. These beta keys are rate limited to a maximum of 42 search requests an hour and 1000 search requests a day. When your app is ready to go live with higher traffic, we highly recommend that you upgrade your beta key to production.

Do I need an API Key in order to use the GIPHY SDK?

Yes. You can obtain an individual rate-limited key by logging into this developer portal with your GIPHY account and then creating an app. You'll be asked to create a GIPHY account if you don't already have one. Please select the SDK box. Once you've created an app, it will be assigned a key which you can use to query our API. If you are planning on having an Android and iOS (or web) implementation, please obtain separate keys for each integration.

How do I upgrade my beta key to Production?

Click on the “Request a production key” link on the app dashboard. We’ll ask you to provide a little bit more info about your app and, if it meets our criteria, we’ll upgrade your app and work with you to make sure your GIPHY integration is rock-solid.
